One-Time Pad (OTP) Algorithm

AIM

To Encrypt and Decrypt the message using One Time Pad Algorithm


EXAMPLE:


Plaintext: HI
Binary of H=01001000
Binary of I=01001001

Keyword: OK
Binary of O=01001111
Binary of K=01001011

Ciphertext:
00000111 = 07(H)
00000010 = 02(A)

Ciphertext: HA
